### [3D Portfolio](https://github.com/FaridDevU/Portafolio-3D-FaridDevU)
Personal portfolio built with Three.js and React Three Fiber. Features immersive 3D navigation, custom animations, and a fully functional Windows 11 interface simulated in the browser — demonstrating what is possible when web development is pushed to its limits.

### [Vertex Hub](https://github.com/FaridDevU/Vertex-Hub)
Immersive 3D gallery built for dynamic visual storytelling. Combines Three.js, Framer Motion, and custom shader work to create a high-end browsing experience.

### [3D Tuning Car](https://github.com/FaridDevU/3D-Tuning-Car)

Pixel-accurate recreation of Apple's iPhone landing page using React and Tailwind CSS. Focused on UI/UX fidelity, micro-animations, and smooth scroll behavior.
